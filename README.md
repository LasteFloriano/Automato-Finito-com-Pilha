Dependências externas:
- Graphviz (necessário para geração de gráficos)
  - Linux: sudo apt install graphviz
  - macOS: brew install graphviz (Homebrew) | sudo port install graphviz (MacPorts)
  - Windows: https://graphviz.org/download/


Para executar:
- python main.py gui
- python main.py (rodar pelo terminal)
